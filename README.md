An ESP32-based Tic-Tac-Toe game with Blynk IoT integration featuring:
- Real-time multiplayer via Blynk app
- FreeRTOS multi-tasking architecture
- LCD display with game status
- Move history and rollback functionality
- Score tracking and game state management
